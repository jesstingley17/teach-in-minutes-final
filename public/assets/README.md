# Logo Assets

Place your logo image file here as `logo.png`.

**Supported formats:**
- PNG (recommended for transparency)
- SVG (scalable vector graphics)
- JPG/JPEG

**File location:**
- Place your logo file at: `public/assets/logo.png`
- The app will automatically use it in the sidebar header
- If the image doesn't load, it will fall back to the emoji icon

**Recommended dimensions:**
- Width: 200-400px
- Height: Auto (maintain aspect ratio)
- Format: PNG with transparent background (if applicable)

**To update the logo:**
1. Replace the file at `public/assets/logo.png` with your new logo
2. The app will automatically pick up the new logo on refresh
